# Base‑Band Tracking Generator Module (0–2.9 GHz)

The Base‑Band module is the fundamental‑mode RF mixing stage of the Tracking Generator.  
It reconstructs the analyzer’s 0–2.9 GHz sweep by combining:

- the analyzer’s 1st LO  
- the analyzer’s 3.1 MHz IF (used internally by the analyzer)  
- a TG‑generated fixed LO_TG from a MAX2870  
- a gain‑boosted LO path to meet the mixer’s 13 dBm LO requirement  
- a fundamental mixer that produces the TG output directly at RF_SA

All frequencies remain coherent with the analyzer because they are derived from the analyzer’s 1st LO and 10 MHz reference.

---

## Fundamental‑mode coherence

In the Base‑Band range, the HP 8562A/B uses fundamental mixing.  
The analyzer’s internal IF is 3.1 MHz, so the analyzer’s 1st LO follows:  LO_SA = RF_SA + 3.1 MHz

For the Base‑Band TG range:  RF_SA = 0–2.9 GHz

Therefore:  LO_SA = RF_SA + 3.1 MHz

This LO_SA is fed directly into the Base‑Band TG mixer as the RF input.

To reconstruct RF_SA, the TG uses a fixed LO_TG generated by the MAX2870:  LO_TG = 3.9107 GHz

The mixer output follows:  RF_TG = LO_SA – LO_TG

Substituting LO_SA = RF_SA + 3.1 MHz:  RF_TG = (RF_SA + 3.1 MHz) – 3.9107 GHz

The TG firmware and calibration ensure that this difference produces:  RF_TG = RF_SA

This is the fundamental‑mode coherence mechanism.

---

## Mixer output and TG coherence

The Base‑Band mixer produces: RF_TG = LO_SA – LO_TG

With:

- LO_SA = RF_SA + 3.1 MHz  
- LO_TG = 3.9107 GHz (after gain stage to 13 dBm)

The TG output becomes:

RF_TG = RF_SA

This guarantees that the TG output tracks the analyzer’s displayed frequency exactly.

---

## Module architecture and building blocks

### Analyzer 1st LO input
- Input: LO_SA = RF_SA + 3.1 MHz  
- Purpose: provides the sweep‑synchronous RF input to the mixer  
- Ensures perfect coherence with analyzer tuning

### MAX2870 LO_TG generator
- Output: LO_TG = 3.9107 GHz (before gain stage)  
- Locked to analyzer’s 10 MHz reference  
- Purpose: provides the LO needed for fundamental mixing

### LO gain stage (to 13 dBm)
- Boosts MAX2870 output to the required mixer LO level  
- Ensures stable convrsion gain  
- Maintains low added phase noise  
- Provides clean 50 Ω drive to the mixer LO port

### HMC219 Fundamental mixer
- RF port: analyzer 1st LO  
- LO port: LO_TG at 13 dBm  
- IF/RF output: 0–2.9 GHz  
- Purpose: produces the Base‑Band TG output via LO_SA – LO_TG

### Low‑pass filter (DC–3 GHz)
- Removes mixer images and LO feedthrough  
- Ensures clean 0–2.9 GHz output

### RF gain stage + output pad
- Provides ~0 dBm output  
- Maintains linearity across the sweep  
- Ensures stable 50 Ω output at the SMA connector

## Performance targets

### Frequency performance
- Output band: 0–2.9 GHz  
- Must track analyzer sweep exactly  
- Fast settling to follow analyzer sweep speed

### Amplitude performance
- Target output: ~0 dBm  
- Flatness corrected at system level  
- No compression across sweep

### Spectral purity
- Low LO_TG leakage  
- Low mixer image content  
- Spurs dominated by MAX2870  
- Phase noise dominated by analyzer LO

### Implementation constraints
- Clean RF layout for DC–3 GHz  
- Good isolation between LO_TG and RF paths  
- Stable operation across temperature and supply variations
